# 💍 혀나곤듀와 새부리의 결혼준비

행복한 결혼을 위한 준비 과정을 체계적으로 관리하는 웹 애플리케이션입니다.

## 🌟 주요 기능

- ✅ 시기별 결혼 준비 체크리스트
- 📊 실시간 진행률 표시
- 💾 로컬 스토리지 자동 저장
- 📱 반응형 디자인 (모바일/태블릿/데스크톱)
- 🎨 아름다운 웨딩 테마 UI

## 🚀 시작하기

### 개발 환경 실행

```bash
npm install
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 확인하세요.

### Vercel 배포

1. [Vercel](https://vercel.com)에 가입/로그인
2. GitHub/GitLab/Bitbucket에 프로젝트 푸시
3. Vercel에서 "Import Project" 클릭
4. 저장소 선택 후 배포

또는 Vercel CLI 사용:

```bash
npm install -g vercel
vercel
```
